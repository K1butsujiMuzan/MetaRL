# Meta Record Label Website

A modern, responsive website for Meta Record Label built with contemporary frontend technologies and best practices.

## 🛠 Tech Stack

**Frontend:**
- React with latest features
- TypeScript for type safety
- Tailwind CSS for styling
- Vite for build tooling

**Key Libraries:**
- React Focus Lock for accessibility
- Custom hooks and components

## 🚀 Quick Start

### Prerequisites
- Node.js or Bun
- Package manager of your choice

### Installation

```bash
# Clone the repository
git clone https://github.com/K1butsujiMuzan/MetaRL.git
cd MetaRL

# Install dependencies
bun install

# Start development server
bun dev
